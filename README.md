# odin-recipes

This project is a website containing several food recipes: grilled cheese,
PBJ and banana, and apple slices.

This project demonstrates basic HTML knowledge. It demonstrates knowledge of
how to write a simple HTML document including boilerplate code and it
demonstrates knowledge of basic HTML tags. Specifically, it uses multiple
levels of header tags, paragraph tags, anchor tags, image tages, and both
ordered and unordered list tags.
